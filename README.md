## Challenge LiterAlura

### Descripción
Aplicación de consola desarrollada en Java que permite buscar libros desde una API pública (Gutendex API) y almacenarlos en una base de datos local para realizar consultas sobre autores, idiomas y registros de libros.

### Funcionalidades
* Buscar libro por título: Consulta la API de Gutendex
* Guarda el libro y sus autores en la base de datos si no existen
* Listar libros registrados: Muestra todos los libros almacenados localmente
* Listar autores registrados: Muestra los autores guardados en la base de datos
* Listar autores vivos en un determinado año: Filtra autores según su fecha de nacimiento y fallecimiento
* Listar libros por idioma: Permite consultar libros según el idioma registrado

### Tecnologías utilizadas
* Java 17, Spring Boot
* PostgreSQL
